# Student Management System

A console-based Student Management System developed in C++ using file handling and menu-driven operations.

## Key Features
- Add new student records (roll number, name, age, department, marks)
- Update existing student information
- Delete student records by roll number
- Display all student records in a formatted table
- Search and display a single student by roll number
- File handling used to store data persistently in students.txt
- Menu-driven interface for smooth, easy navigation
- Input validation to handle invalid/non-numeric entries

## Expected Outcome
